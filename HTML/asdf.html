b 태그는 안 먹을 수도 있는데 그럼 strong 태그를 쓰라고 합니다.
? & 로 구분지을 수 있는 parameter
#fragment는 스크롤로 챕터를 나누는 느낌
port는 해당 사이트를 한정된 사람들만 접속시키고 싶을 떄 새롭게 만드는 주소 느낌(문)

http://example.com:80/toys?p=foo#hello
Host = example.com
Protocol = http
Fragment = #hello
Query = p=foo
Port = 80
Path = toys

request line - 주소를 치면 브라우저가 리퀘스트 라인을 보냄.
GET(METHOD) /(PATH) HTTP/1.1
http://example.com:80/toys?p=foo#hello --> GET /toys?p=foo HTTP/1.1
request(request line) <-> response(Status line)
ex) HTTP/1.1(Version) 200(Status code) OK(Reson phrase)
200 - 됨, 404 - 파일 찾지 못함, 500 - 서버 없음, 302 - 다른데에 위치함
보통 request line이나 status line을 보낸 다음 Followed by Headers라고 하는 다양한 형식의 정보들을 보낸다.
